História da Web

Primeiros Computadores

Primeira geração

Harvard Mark I (1944)baseado em válvulas, eletromecânico

Colossus (1946) - criado para criptoanálises, contexto da Segunda Guerra Mundial (Alemanha) ENIAC (1946) - criado pelos Estados Unidos, mais automático

Segunda geração (1959-1965)

Evolução das válvulas para os transistores Computadores mais portáteis

Terceira Geração (1965 a 1970)

Circuitos integrados, computadores menores

Quarta geração (a partir de 1971)

Computadores pessoais, ainda menores, mais portáteis e acessiveis Empresas como Apple e Microsoft passam a ter grande importancia

Quinta geração

Computadores quânticos, inteligência artificial, metaverso, criptomoedas, etc.

Surgimento da Web

1958 - Telegráfo, cabeamento intercontinental 1950 - Guerra Fria, desenvolvimento da tecnologia Importância da teoria de Joseph Carl Licklider, a respeito de uma rede onde todos podem se comunicar através dela 1969 - Arpanet - um sistema de comunicacao entre os extremos dos Estados Unidos TCP/IP 1989 - World Wide Web - Popularização da tecnologia

Client Side

É o lado caracterizado pelos dispositivos utilizados pelo usuário para se conectar à Web. É composto pelas interfaces do usuário.

Navegadores

Programas utilizados para abrir/executar os arquivos do servidor Web. Ex: Firefox, Chrome, Opera, etc.

Aplicações Web

São programas que podem ser acessados atraves da internet, tendo ela como meio de comunicacao entre o cliente e o servidor, não necessitando de instalação.

Dispositivos móveis

São tecnologias utilizadas para acessar a internet, apps e outros recursos, de forma portatil.

Linguagens para clients (Front-End)

HTML CSS Javascript

Server Side

É o lado que processa e armazena as informações requisitadas pelos clientes. Podem ser computadores simples ou grandes e robustos, onde arquivos de diversos clientes sao armazenados.

Tipos de servidores

Alguns exemplos de servidores sao os de armazenamento de arquivos, os responsaveis pela seguranca, os de streaming de conteudos, os responsaveis por executar funcoes na internet, etc.

Linguagens para servidores (Back-End)

PHP Javascript C# Javascript Ruby
