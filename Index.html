História da Web

Primeiros Computadores

Primeira geração

  1. Mark I (1944)baseado em válvulas, eletromecânico
  2. Colossus (1946) - criado para criptoanálises, contexto da Segunda Guerra Mundial (Alemanha)
  3. ENIAC (1946) - criado pelos Estados Unidos, mais automático

Segunda geração (1959-1965)

Evolução das válvulas para os transistores Computadores mais portáteis

Terceira Geração (1965 a 1970)

Circuitos integrados, computadores menores

Quarta geração (a partir de 1971)

Computadores pessoais, ainda menores, mais portáteis e acessíveis Empresas como Apple e Microsoft passam a ter grande importância

Quinta geração

Computadores quânticos, inteligência artificial, metaverso, criptomoedas, etc.

Surgimento da Web

  • 1958 - Telegráfo, cabeamento intercontinental
  • 1950 - Guerra Fria, desenvolvimento da tecnologia
  • 1962 - Importância da teoria de Joseph Carl Licklider, a respeito de uma rede onde todos podem se comunicar através dela
    • "In a few years, men will be able to communicate more effectively through a machine than face to face. That is a rather startling thing to say, but it is our conclusion."
  • 1969 - Arpanet - um sistema de comunicação entre os extremos dos Estados Unidos
  • 1989 - World Wide Web - Popularização da tecnologia>

Client Side

É o lado caracterizado pelos dispositivos utilizados pelo usuário para se conectar à Web. É composto pelas interfaces do usuário.

Navegadores

Programas utilizados para abrir/executar os arquivos do servidor Web. Ex: Firefox, Chrome, Opera, etc.

Aplicações Web

São programas que podem ser acessados através da internet, tendo ela como meio de comunicação entre o cliente e o servidor, não necessitando de instalação.

Dispositivos móveis

São tecnologias utilizadas para acessar a internet, apps e outros recursos, de forma portátil.

Linguagens para clients (Front-End)

  1. HTML
  2. CSS
  3. Javascript

Server Side

É o lado que processa e armazena as informações requisitadas pelos clientes. Podem ser computadores simples ou grandes e robustos, onde arquivos de diversos clientes são armazenados.

Tipos de servidores

Alguns exemplos de servidores são os de armazenamento de arquivos, os responsáveis pela seguranca, os de streaming de conteúdos, os responsáveis por executar funcoes na internet, etc.

Linguagens para servidores (Back-End)

  1. PHP
  2. Javascript
  3. C#
  4. Javascript
  5. Ruby
